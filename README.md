> file vs database

    - 파일에 대한 것들을 다루기가 어렵다
    - 스프레드시트 구조 작성
    - file -> spreadsheet (구조적)
    - file -> spreadsheet -> database
    - 프로그램적으로 자동화가 가능
    - 구조적으로 데이터를 저장을 하면 데이터의 가공이 쉽다.
